 Expense Tracker Dashboard

SmartSpend is an animated expense tracking web application that helps users manage income and expenses easily.

## Features
- Add income and expenses
- Category selection
- Dashboard cards
- Balance calculation
- Delete transactions
- Animated glassmorphism UI
- Responsive layout

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Categories
- Food
- Shopping
- Travel
- Education
- Entertainment

## How to Run
1. Open project in VS Code
2. Open `index.html`
3. Run using Live Server
